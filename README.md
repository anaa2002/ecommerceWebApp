# MoonCart

A full-stack MERN e-commerce app with authentication, admin product management, cart functionality, Stripe checkout, Redis caching, and Cloudinary image support.

## Features

- User signup, login, logout
- Cookie-based authentication
- Protected admin dashboard
- Create, update, delete products
- Feature/unfeature products
- Featured/recommended/category product sections
- Shopping cart
- Stripe checkout
- Order creation after successful payment
- Cart clears after purchase

## Tech Stack

React, Node.js, Express, MongoDB, Redis, Stripe, Cloudinary

## Setup

Create `.env` files using the provided `.env.example` files.

Install dependencies:

```bash
cd backend
npm install

cd ../frontend
npm install
```
